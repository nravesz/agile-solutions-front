# How to run (with back)

## 1) Run the server
```
git clone https://github.com/nravesz/agile-solutions-back.git
cd agile-solutions-back
npm install
node index.js
```

## 2) Run the front
```
https://github.com/nravesz/agile-solutions-front.git
cd agile-solutions-front
git checkout get
npm install
npm start
```
